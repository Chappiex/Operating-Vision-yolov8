# Operating-Vision-yolov8
the training and data collection code for my operating vision model. 
The model is a 15 class model wioth the classes being:
backhoe_loader
bucket
bucket_empty
bucket_full
bulldozer
dump_truck
excavator
grader
person_no_ppe
person_ppe
road_roller
safety_barrel
safety_cone
trench
wheel_loader
You can download the mode here: https://huggingface.co/Chappieut/Industrial-Site-Safety-Detection-v1 
You can download the data here: https://huggingface.co/datasets/Chappieut/Industrial-Site-Safety-Detection-v1-DATASET
